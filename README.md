# SAP Fiori Purchase Order Approval App

## Business Scenario
This app simulates a Purchase Order approval workflow in SAP S/4HANA.

## Technical Architecture
- Frontend: SAP UI5 (MVC)
- Backend: OData V2 (Mock Service)
- Deployment: SAP BAS
- Launchpad Configuration included

## Features
- PO List (Filterable)
- Vendor Details
- Status-based Approval
- Value Help Dialog
