## 1. Business Scenario

This SAP UI5 application simulates a Purchase Order approval workflow
in an S/4HANA-like environment.

The app enables users to:

- View purchase orders
- Filter and search POs
- Display vendor details
- Approve or reject purchase orders
- Use value help dialogs

The goal of this project is to demonstrate hands-on expertise in
SAP UI5, Fiori Elements, and OData integration.

---

## 2. Technical Architecture

Frontend:
- SAP UI5
- Fiori Elements (List Report + Object Page)
- MVC Architecture
- JavaScript (ES6)
- XML Views

Backend Simulation:
- OData V2 service (mocked structure)
- SAP Gateway style integration
- CDS-ready structure

Deployment:
- SAP Business Application Studio (BAS)
- Git version control
